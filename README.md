## Overview

VO Audition Tracker is aimed at voiceover professionals who need a more orderly way to follow audition activity and keep performance-related information in one place. The dashboard format makes it easier to inspect data inside the browser instead of juggling separate notes, files, or scattered references.

It is a practical fit for anyone handling auditions on a regular basis and looking for a straightforward workspace to monitor progress. By pairing audition organization with metric tracking, the project helps you compare entries, review trends, and maintain records in a cleaner structure.
